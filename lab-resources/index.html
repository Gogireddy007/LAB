Lab Resources

This page collects resources that are useful for lab members and interested researchers.

Computational Resources

  • GitHub Organization: Lab code repositories, scripts, and pipelines
  • R for Data Science: Essential reference for data analysis in R
  • Bioinformatics Workbook: Practical guides for common bioinformatics tasks

Data & Databases

  • NCBI GenBank: Nucleotide sequence database
  • IMG/M: Integrated Microbial Genomes & Microbiomes
  • SILVA rRNA Database: Ribosomal RNA gene sequences

Writing & Communication

  • Grant Writing: NSF BIO, OCE, and EAR programs are most relevant to lab work
  • Journal Recommendations: AEM, ISME Journal, Frontiers in Microbiology, Biogeosciences

For Incoming Lab Members

If you are joining the Steen Lab, please:

  1. Set up your UTK network credentials and VPN
  2. Request access to our shared computational resources (contact Drew)
  3. Review our lab GitHub organization and clone relevant repos
  4. Familiarize yourself with R and the tidyverse package ecosystem
  5. Read key papers listed on our Publications page

Protocols

Internal lab protocols are available to lab members via our shared drive. Contact Drew for access.
